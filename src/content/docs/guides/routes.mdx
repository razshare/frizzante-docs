You can add a new route to the server with `AddRoute()`.

```go
//main.go
package main

import (
    "github.com/razshare/frizzante/frz"
    "main/lib/handlers"
)

var hello = frz.Route{
    Pattern: "GET /", 
    Handler: handlers.Hello,
}

func main() {
    frz.NewServer().
        AddRoute(hello).    // Adds route.
        Start()
}
```

```go
//lib/handlers/hello.go
package handlers

import "github.com/razshare/frizzante/frz"

func Hello(c *frz.Connection) {
    c.SendMessage("Hello")    // Sends text.
}
```
