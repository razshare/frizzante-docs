title: Creating & Starting a Server
---

import { FileTree} from '@astrojs/starlight/components';

Use `NewServer()` to create a server then `Start()` it.


```go
package main

import (
	"embed"
	"github.com/razshare/frizzante/frz"
)

//go:embed .dist/*/**
var dist embed.FS

func main() {
    frz.NewServer().
        WithDist(dist).
        Start()
}
```
