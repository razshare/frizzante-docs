This document is a guide on how to submit bug reports and feature requests.

## Reporting a Bug

Make sure to read the [documentation](https://razshare.github.io/frizzante-docs/) before submitting a bug report.

Bug reports can be submitted using the [GitHub issue tracker](https://github.com/razshare/frizzante/issues/new?template=bug_report.md).

When reporting a bug, make sure to include a link to a repository isolating the issue and the steps to reproduce said issue.

## Requesting a Feature

Make sure to read the [documentation](https://razshare.github.io/frizzante-docs/) before submitting a feature request.

Feature request can be submitted using the [GitHub issue tracker](https://github.com/razshare/frizzante/issues/new?template=feature_request.md).
