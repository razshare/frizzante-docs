---
title: Type Definitions
---

import { Steps } from '@astrojs/starlight/components'

It is possible, but not required, to generate TypeScript type definitions from Go structs with `types.Generate[T]()`, where `T` is the type you wish to generate.


<Steps>
1. Define your Go types.
    ```go
    //lib/routes/handlers/welcome/view.go
    package welcome

    import (
        "main/lib/core/client"
        "main/lib/core/send"
        "main/lib/core/view"
    )

    type Props struct {
        Message string
        Error   string
    }

    func View(client *client.Client) {
        send.View(client, view.View{
            Name: "Welcome",
            Props: Props{
                Message: "hello",
            },
        })
    }
    ```

    :::note
    Struct `json` tags currently ignored.
    :::

1. Create a new `lib/types/main.go` file and define the `main()` function in it.
    ```go
    //lib/types/main.go
    package main

    func main() {}
    ```
1. Call `types.Generate[T]()` in the `main()` function.
    ```go
    //lib/types/main.go
    package main

    import (
        "main/lib/core/types"
        "main/lib/routes/handlers/welcome"
    )

    func main() {
        types.Generate[welcome.Props]()
    }
    ```
1. Generate type definitions.
    ```sh
    frizzante -gtypes
    ```
    This will generate your type definitions in `app/types/gen`.

    ```ts
    //app/types/gen/main/lib/routes/handlers/welcome/Props.d.ts
    export type Props = {
        Message: string
        Error: string
    }
    ```
</Steps>
