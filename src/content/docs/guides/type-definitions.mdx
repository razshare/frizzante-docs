---
title: Type Definitions
---

import { Steps } from '@astrojs/starlight/components'

It is possible, but not required, to generate TypeScript type definitions from Go structs with `types.Generate[T]()`, where `T` is the type you wish to generate.


<Steps>
1. Define your Go types.
    ```go
    //lib/routes/handlers/welcome/view.go
    package welcome

    import (
        "main/lib/core/client"
        "main/lib/core/send"
        "main/lib/core/view"
    )

    type Props struct {
        Message string `json:"message"`
        Error   string `json:"error"`
    }

    func View(client *client.Client) {
        send.View(client, view.View{
            Name: "Welcome",
            Props: Props{
                Message: "hello",
            },
        })
    }
    ```
    
    :::tip
    Struct `json` tags are not required, but if present, they will be taken into account when generating type definitions.
    :::

1. Create a new `types.go` file and define the `init()` function in it.
    ```go
    //types.go
    //go:build types

    package main

    func init() {}
    ```

    :::caution
    Make sure to exclude this file from the build process by tagging it with `//go:build types`.
    :::
1. Call `types.Generate[T]()` in the `init()` function.
    ```go
    //types.go
    //go:build types

    package main

    import (
        "main/lib/core/types"
        "main/lib/routes/handlers/welcome"
    )

    func init() {
        types.Generate[welcome.Props]()
    }
    ```
1. Generate type definitions.
    ```sh
    frizzante -gtypes
    ```
    This will generate your type definitions in `app/types/gen`.

    ```ts
    //app/types/gen/main/lib/routes/handlers/welcome/Props.d.ts
    export type Props = {
        message: string
        error: string
    }
    ```
</Steps>
