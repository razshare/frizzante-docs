---
title: Type Definitions
---

import { Steps } from '@astrojs/starlight/components'

It is possible, but not required, to generate TypeScript type definitions from Go structs with `types.Generate[T]()`, where `T` is the type you wish to generate.


<Steps>
1. Define your Go types.
    ```go
    //lib/routes/welcome/types.go
    package welcome

    type Props struct {
        Message string `json:"message"`
        Error   string `json:"error"`
    }
    ```

    :::note
    Struct `json` tags are optional.
    :::
1. Call `types.Generate[T]()`.
    ```go
    //lib/routes/welcome/types.go
    package welcome

    func init() {
        _ = types.Generate[Props]()
    }

    type Props struct {
        Message string `json:"message"`
        Error   string `json:"error"`
    }
    ```
1. Generate types

    ```sh
    frizzante -gtypes
    ```

    :::tip
    Alternatively, starting dev mode will also generate types.
    ```sh
    frizzante --dev # this will automatically generate types while you modify them
    ```
    :::

    

    This will generate your type definitions in `.gen/types`.

    ```ts
    //.gen/types/main/lib/routes/welcome/Props.d.ts
    export type Props = welcome.Props

    export declare namespace welcome {
        export type Props = {
            message: string
            error: string
        }
    }
    ```

    :::tip
    You can use the default `$gen` alias to access the `.gen` directory.
    ```svelte
    <script lang="ts">
        import type { Props } from "$gen/types/main/lib/routes/welcome/Props"
        let { message, error }:Props = $props()
    </script>
    ```
    :::
</Steps>
