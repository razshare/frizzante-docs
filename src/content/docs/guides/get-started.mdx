Frizzante is an opinionated web server framework written in Go that uses Svelte to render web pages.

#### Prerequisites

Download the latest frizzante binary [from github](https://github.com/razshare/frizzante/releases).

:::tip
Or build it yourself
```sh
git clone https://main/lib/core --depth=1
cd frizzante && make zip && go build -o frizzante
```
:::

:::tip
Remember to add the binary to your path.

```sh
export PATH="/path/to/frizzante:$PATH"
```
:::

#### Get Started

<Steps>
1. Create project.

    ```sh
    frizzante -c MyProject
    ```

1. Configure project.

    ```sh
    frizzante --configure
    ```

1. Start development.

    ```sh
    frizzante --dev
    ```

1. Build.

    ```sh
    frizzante --build
    ```

    This will create a `.gen/bin/app` standalone executable.

    <FileTree>
       - .gen
          - bin
             - app
    </FileTree>
</Steps>

:::tip
You can also choose cli options interactively by just running `frizzante` without any flags.
:::
