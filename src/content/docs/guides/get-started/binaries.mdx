#### Get Started

<Steps>
1. Create a project.

    ```sh
    frizzante -c MyProject
    ```

1. Configure project.

    ```sh
    make configure
    ```

1. Start development.

    ```sh
    make dev
    ```

1. Build.

    ```sh
    make build
    ```

    This will create a `.gen/bin/app` standalone executable.

    <FileTree>
        - .gen
            - bin
                - app
    </FileTree>
</Steps>
